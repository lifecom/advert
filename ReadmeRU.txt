Плагин Advert предназначен для управления рекламными компания на сайте под управлением livestreet версии 1.0.2 (на других не тестировалось)

Установка : 

1. Скачайте плагин с https://github.com/lsmaka/advert и распакуйте в содержимое в директорую plugins/advert вашего сайта
2. Перед тем как активировать плагин (или сразу после его активации) необходимо выполнить обязательную настройку в 
plugins/advert/config/config.php
	// Пользователь чья реклама будет показываться по умолчанию
	// Укажите реального пользователя в вашей системе реклама которого будет показываться по умолчанию
	$config['defuser'] = 'admin';
3. Можно преступать к созданию рекламных компаний.
